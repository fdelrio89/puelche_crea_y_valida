# Puelche — Postulación Crea y Valida

Espacio de trabajo para la postulación de **Puelche** al fondo **CORFO Crea y Valida (Fase I, 2026-1)**.

Puelche es un sistema basado en agentes de IA que apoya la evaluación de licitaciones públicas en Chile, automatizando el análisis documental y la estructuración de evidencia, manteniendo las decisiones bajo control humano.

## Contenido

- `corfo_crea_y_valida_perfil_de_proyecto.md` — Formulario del perfil de proyecto (documento principal).
- `KB/` — Material de referencia: bases del concurso, formulario y antecedentes de la formulación FONDEF previa.
- `AGENTS.md` — Contexto y convenciones de trabajo para la redacción.
